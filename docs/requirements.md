# AI 驱动的代码生成工具 - 需求文档 v1.0

## 1. 项目概述
一款面向前端开发者的 AI 代码生成工具：用户输入自然语言需求描述，系统自动生成可运行的代码文件（组件、页面、API 等），支持预览、迭代修改和导出。

## 2. 核心目标
- 输入：自然语言需求 + 可选上下文（现有代码、框架偏好、风格指南）
- 输出：可直接运行的代码文件（夹） + 可视化预览
- 交互：支持多轮对话式迭代修改

## 3. 功能需求

### 3.1 需求输入
- Markdown / 纯文本输入框
- 支持粘贴截图（通过多模态模型理解 UI 草图）
- 可选择目标框架（React / Vue / Nuxt / Next 等）
- 可选择语言（TypeScript / JavaScript）
- 支持上传现有代码作为上下文

### 3.2 代码生成
- 调用大模型 API（如智谱 GLM / OpenAI / Claude）生成代码
- 生成结构：单文件组件或多文件项目
- 生成内容包括：组件代码、样式（CSS/Tailwind/SCSS）、类型定义、测试文件
- 支持生成完整页面 + 路由配置

### 3.3 代码预览
- 内置 iframe / WebContainer 实时预览
- 展示生成的文件树和文件内容
- 语法高亮显示 Diff 对比

### 3.4 迭代修改
- 多轮对话：对生成结果提出修改意见
- 支持在生成代码中手动编辑后继续对话
- 历史版本回溯

### 3.5 导出
- 下载单个文件 / 整个项目 ZIP
- 一键复制代码
- 可选：直接推送到 GitHub / 本地目录

## 4. 技术架构

```
┌─────────────────────┐
│    Frontend (React)  │  ← 用户界面
├─────────────────────┤
│    Agent Layer       │  ← AI 编排（工具调用、上下文管理）
├─────────────────────┤
│    Backend (Node)    │  ← API 代理、文件操作
├─────────────────────┤
│    LLM API           │  ← 智谱 / OpenAI / Claude
└─────────────────────┘
```

### 4.1 前端（React + TypeScript）
- 编辑器：Monaco Editor / CodeMirror
- 预览：Sandpack / WebContainer
- UI：Tailwind CSS + shadcn/ui
- 状态管理：Zustand

### 4.2 Agent 层
- 使用 MCP 协议连接 LLM 与工具
- 工具：文件读写、命令行执行、代码分析
- 上下文管理：对话历史 + 项目状态 tree

### 4.3 后端（Node.js + Express/Hono）
- 代理 LLM API 请求
- 文件系统操作
- 项目打包下载

## 5. 非功能需求
- 生成平均耗时 < 30s（流式输出）
- 支持主流前端框架（React / Vue / Nuxt / Next）
- Token 消耗透明显示
- 代码安全：沙箱执行预览

## 6. MVP 范围
第一期做精简版：
- React 组件生成（TS + Tailwind）
- 单文件预览
- 多轮修改
- 复制 / 下载
